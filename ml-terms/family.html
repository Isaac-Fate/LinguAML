Families of Machine Learning Models to Learn#

from linguaml.tolearn.family import Family

Type of the hyperparameter configuration of a model family can be accessed by:

Family.SVC.hp()

linguaml.tolearn.family.svc.SVCConfig

To create a model instance, use the method deifne_mdoel.

You can create it by providing the hyperparameters as keyword arguments to the model class:

Family.SVC.define_model(
    C=10,
    kernel="linear"
)

SVC(C=10, kernel='linear')
